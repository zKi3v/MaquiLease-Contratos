# 📦 MaquiLease - Gestión Inteligente de Maquinaria

Sistema web premium para la gestión, alquiler y venta de maquinaria industrial. Diseñado para optimizar la logística y el control transaccional con una capa de inteligencia de negocios.

## 🚀 Estado del Proyecto
Actualmente el proyecto se encuentra en la **Fase de Core Transaccional y UI Modernizado**.

*   **Frontend**: Desplegado en [Vercel](https://maquilease.vercel.app).
*   **Diseño**: Estilo corporativo minimalista (Dark/Light mode).
*   **Entornos**: Sistema dual (Test con datos ficticios / Producción).

## 📂 Estructura del Proyecto
*   `/frontend`: Aplicación Angular 18 (Standalone Components).
*   `/backend`: API REST en .NET 7.
*   `/docker`: Configuración para entorno de pruebas local.
